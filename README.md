# paralyzed_bind

### Opis

Po rozpoczeciu czynnosci, ktora paralizuje ruch, uruchamia sie **functional bind** na `przestan`. Edycja ustawia bind na `~q`

W czasie paralizu dolny bar podswietla sie na czerwono.
